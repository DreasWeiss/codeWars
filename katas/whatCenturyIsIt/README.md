<p>Return the century of the input year. The input will always be a 4 digit string, so there is no need for validation. </p>
<h3 id="examples">Examples</h3>
<pre><code>"1999" --&gt; "20th"
"2011" --&gt; "21st"
"2154" --&gt; "22nd"
"2259" --&gt; "23rd"
"1124" --&gt; "12th"
"2000" --&gt; "20th"
</code></pre>
